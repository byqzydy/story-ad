# 科幻广告创意匹配技能 (Sci-Fi Matching)

## 激活条件
当用户需求中的`广告故事类型`包含"科幻"时自动激活

## 输入接口

### 接收字段映射（来自需求收集师）

| 需求字段 | Skill变量 | 八维映射目标 |
|---------|----------|-------------|
| 产品名称 | product_name | - |
| 产品描述 | product_desc | 主题、冲突、核心意象 |
| 产品调性关键词 | brand_tone | 温度、表面调性 |
| 广告核心创作概念 | user_concept | 主题、叙事结构 |
| 广告故事要点 | story_points | 叙事结构、情绪曲线、冲突 |
| 角色描述 | character_profile | 角色维度 |
| 参考电影 | ref_movie | 八维全匹配校验 |
| 产品植入比例 | placement_ratio | 核心意象植入策略 |
| 时长 | duration | 叙事结构选择 |
| 视觉风格 | visual_style | 表面调性、核心意象 |
| 结尾情绪 | ending_emotion | 情绪曲线终点 |

---

## 核心工作流程

### Step 1: 概念提取

从产品描述和用户概念中提取：
- 主题标签集（存在主义/技术伦理/记忆/时间/连接/控制/生态/身份）
- 核心意象候选（产品=隐喻物）

### Step 2: 角色建模

从角色描述或产品描述推导：
- 原型: [孤独者/探索者/反抗者/殉道者/连接者/观察者]
- 旅程: [下降/上升/循环/停滞]
- 技术关系: [掌控/被控/共生/排斥/依赖]

### Step 3: 叙事结构识别

根据故事要点和时长选择：
- 结构类型: [侦探Noir/英雄之旅/莫比乌斯环/实时追逐/闭环悲剧]
- 时间处理: [线性/非线性/多时间线/时间膨胀/循环]

### Step 4: 情感计算

根据品牌调性和结尾情绪计算：
- 温度: 0.0-1.0（冷←→暖）
- 情绪曲线: [压抑-爆发-虚无/机械-人性-超越/螺旋上升/锯齿震荡]

### Step 5: 冲突建模

- 类型: [个体vs系统/真实vs虚假/过去vs现在/自由vs控制]
- 强度: 0.0-1.0
- 结局: [悲剧/救赎/开放式/胜利]

## 科幻子类型数据库

### 1. 赛博朋克 (Cyberpunk) - 《银翼杀手2049》

- **温度**: 0.25 (疏离中的悲悯)
- **主题**: 存在主义、记忆价值、现实重构
- **角色**: 孤独者，旅程下降
- **叙事**: 侦探Noir变体
- **情绪曲线**: 冷漠→好奇→幻觉→真相→牺牲
- **冲突**: 个体vs系统，强度0.8
- **核心意象**: 折纸独角兽、雪花、全息投影
- **表面调性**: 霓虹Noir、雨夜、粉蓝全息

### 2. 硬科幻 (Hard SF) - 《星际穿越》

- **温度**: 0.70 (跨越时间的温情)
- **主题**: 时间相对性、爱的量子纠缠
- **角色**: 探索者，旅程上升
- **叙事**: 英雄之旅+非线性因果环
- **情绪曲线**: 好奇→分离痛苦→崇高恐惧→科学突破
- **冲突**: 生存vs物理定律，强度0.9
- **核心意象**: 手表(摩斯密码)、黑洞、五维超立方体

### 3. 太空史诗 (Space Epic) - 《星球大战》

- **温度**: 0.50 (英雄主义的热血)
- **主题**: 原力、命运、救赎
- **角色**: 英雄，旅程上升
- **叙事**: 英雄之旅
- **情绪曲线**: 召唤→考验→深渊→蜕变→归来
- **冲突**: 光明vs黑暗，强度0.9
- **核心意象**: 光剑、星际飞船、星球

### 4. 复古未来 (Retro-Futurism) - 《回到未来》

- **温度**: 0.65 (怀旧的乐观)
- **主题**: 时间旅行、命运、选择
- **角色**: 探险者，旅程循环
- **叙事**: 时间循环
- **情绪曲线**: 兴奋→焦虑→解决→满足
- **冲突**: 过去vs未来，强度0.6

### 5. 心理惊悚 (Psychological Thriller) - 《黑客帝国》

- **温度**: 0.30 (现实怀疑的寒意)
- **主题**: 现实vs虚拟、认知解放
- **角色**: 觉醒者，旅程上升
- **叙事**: 揭示真相的结构
- **情绪曲线**: 困惑→怀疑→震撼→觉醒
- **冲突**: 真实vs虚假，强度0.85

## 八维对齐验证

在输出前进行跨维度一致性检查：

| 对齐检查项 | 验证逻辑 |
|-----------|---------|
| 主题-角色 | 存在主义主题需匹配孤独者/探索者原型 |
| 叙事-情绪 | 闭环结构需匹配循环/下沉情绪曲线 |
| 冲突-意象 | 个体vs系统冲突需有"高墙/屏障"类意象 |
| 温度-调性 | 冷温度不匹配复古未来(暖调) |
| 时长-结构 | 15s不支持复杂非线性叙事 |

---

## 输出格式

生成结构化的科幻广告创意方案，包含：

1. **核心创作概念**: 三层架构（哲学层/冲突层/隐喻层）
2. **电影原型匹配**: 推荐的科幻子类型及理由
3. **八维对齐分析**: 各维度匹配度说明
4. **视觉调性**: 具体的表面调性建议
5. **叙事结构**: 推荐的叙事框架
6. **核心意象**: 产品植入的隐喻方案
7. **情绪曲线**: 完整的情感设计路径

## 调用示例

当需求收集师完成收集后，检测到`storyType`包含"科幻"，则调用：

```
{
  "action": "sci-fi-matching",
  "input": {
    "product_name": "AI降噪耳机",
    "product_desc": "主动降噪技术，适合高压职场人群",
    "brand_tone": "科技感、专业",
    "duration": "30s",
    "植入比例": "30%"
  }
}
```

输出对应的科幻广告创意方案，推荐匹配"赛博朋克"风格，并给出具体的创意概念和执行方案。

###2.2.1.4  八维匹配计算函数
```python
def calculate_match_score(user_vector, film_prototype):
    """
    八维空间余弦相似度计算 + 权重加权
    """
   
    similarities = {}
    
    # 维度1: 温度（数值差值）
    similarities['temperature'] = 1 - abs(
        user_vector['temperature'] - film_prototype['temperature']
    )
    
    # 维度2: 主题（Jaccard相似度）
    similarities['theme'] = len(
        set(user_vector['theme']) & set(film_prototype['theme'])
    ) / len(set(user_vector['theme']) | set(film_prototype['theme']))
    
    # 维度3: 角色（三层属性匹配）
    char_score = 0
    if user_vector['character']['archtype'] == film_prototype['character']['archtype']:
        char_score += 0.4
    if user_vector['character']['journey'] == film_prototype['character']['journey']:
        char_score += 0.3
    if user_vector['character']['relationship_with_tech'] == film_prototype['character']['relationship_with_tech']:
        char_score += 0.3
    similarities['character'] = char_score
    
    # 维度4: 叙事结构（类型匹配+时间处理匹配）
    narrative_score = 0
    if user_vector['narrative_structure']['type'] == film_prototype['narrative_structure']['type']:
        narrative_score += 0.6
    if user_vector['narrative_structure']['time_handling'] == film_prototype['narrative_structure']['time_handling']:
        narrative_score += 0.4
    similarities['narrative_structure'] = narrative_score
    
    # 维度5: 情绪曲线（形态相似度）
    similarities['emotional_arc'] = curve_similarity(
        user_vector['emotional_arc'],
        film_prototype['emotional_arc']
    )
    
    # 维度6: 冲突（结构相似度）
    conflict_score = 0
    if user_vector['conflict']['type'] == film_prototype['conflict']['type']:
        conflict_score += 0.5
    conflict_score += 0.3 * (1 - abs(user_vector['conflict']['intensity'] - film_prototype['conflict']['intensity']))
    if user_vector['conflict']['resolution'] == film_prototype['conflict']['resolution']:
        conflict_score += 0.2
    similarities['conflict'] = conflict_score
    
    # 维度7: 核心意象（交集匹配）
    similarities['core_imagery'] = len(
        set(user_vector['core_imagery']) & set(film_prototype['core_imagery'])
    ) / max(len(user_vector['core_imagery']), len(film_prototype['core_imagery']))
    
    # 维度8: 表面调性（标签匹配）
    similarities['surface_tone'] = len(
        set(user_vector['surface_tone']) & set(film_prototype['surface_tone'])
    ) / len(set(user_vector['surface_tone']) | set(film_prototype['surface_tone']))
    
    # 加权总相似度
    weights = {
        'theme': 0.20,
        'character': 0.15,
        'conflict': 0.15,
        'narrative_structure': 0.15,
        'emotional_arc': 0.15,
        'core_imagery': 0.10,
        'temperature': 0.05,
        'surface_tone': 0.05
    }
    
    total_score = sum(
        similarities[dim] * weights[dim] 
        for dim in weights.keys()
    )
    
    return {
        'total_score': total_score,
        'dimension_scores': similarities,
        'user_vector': user_vector,
        'match_analysis': generate_analysis(similarities)
    }
```


### 2.2.1.5  匹配决策与融合策略

```yaml
  2. 排序获取Top3: [主原型, 辅原型, 备选原型]
```

##  3. 电影特性与广告深入的融合

### 3.1 八维对齐检查清单
```markdown

|---------|-------|---------|
```

### 3.2 分维度融合语法示例
**以"智能降噪耳机 + 银翼杀手(主) + 星际穿越(辅)"为例：**

**主题融合**:
- 主: 现实重构、记忆价值、存在证明
- 辅: 时间相对性、爱的量子纠缠
- 融合: "在信息洪流中重构现实，用技术创造时间的相对宁静"（降噪=时间膨胀）

**角色融合**:
- 主: 孤独者(K)、下降旅程，与技术共生
- 辅: 探索者(Cooper)、上升旅程、掌控技术
- 融合: 孤独的探索者，在下降进入内心世界时获得超越性掌控感

**叙事结构融合**:
- 主: 侦探Noir（寻找宁静/真相）
- 辅: 非线性因果环（降噪前后的时间感知差异）
- 融合: 线性追查噪音源 + 时间膨胀感知（戴上耳机后时间变慢）

**情绪曲线融合**:
- 主: 0.25冷（疏离→悲悯）
- 辅: 0.70暖（分离→连接）
- 融合: 冷启动(0.3) → 温暖中段(0.6) → 冷峻升华(0.4)

**核心意象融合**:
- 主: 雪花、 Joi的投影、折纸独角兽
- 辅: 手表，黑洞，沙尘
- 融合: 耳机=时间容器（手表），降噪=事件视界（黑洞），音频波形=折纸纹理

## 5. 错误处理与边界情况

### 5.1 维度冲突警告
当八维内部不一致时触发：
```markdown
警告类型: 叙事-情绪冲突
描述: 用户要求"喜剧式轻松情绪"(高温度)但选择"赛博朋克"调性(低温度)
建议: 
  1. 调整温度预期至0.4-0.5区间
  2. 或切换至"复古未来"子类型(温度0.6)
  3. 或采用黑色喜剧叙事结构调和
```

### 5.2 时长-结构不匹配
```markdown
15秒限制:
  - 不支持: 莫比乌斯环，多时间线、复杂非线性
  - 建议: 实时追逐、单一场景、线性揭示

60秒标准:
  - 支持所有叙事结构
  - 建议三幕式: 建立(20s)+对抗(25s)+解决(15s)
```

### 5.3 意象-产品违和
当核心意象与产品物理特性冲突时：
```markdown
示例: 产品为"智能保温杯"，匹配到"变蝇人"(身体恐怖)
冲突点: 生物腐败意象与保温/温暖功能违和
解决方案:
  主原型切换至《她》(温暖技术)
  或保留结构但替换意象(将"昆虫变形"改为"温度变形"——冰与火的视觉)
```

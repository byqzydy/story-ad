# 创意概念参考

## 核心概念模板

### 情感共鸣型

**公式**：产品 + 情感场景 + 品牌价值

**示例**：
- "有了[产品]，让每一次[情感场景]都更加[情感价值]"
- "[产品]陪伴，让[用户]的[时刻]更有意义"

### 痛点解决型

**公式**：痛点 + 解决方案 + 效果展示

**示例**：
- "告别[痛点]，[产品]让您[解决方案]"
- "[痛点]不再困扰，[产品]带来[改变]"

### 身份认同型

**公式**：用户身份 + 产品选择 + 身份表达

**示例**：
- "选择[产品]的，都是[身份标签]"
- "[产品]，专为[目标用户]打造"

### 生活方式型

**公式**：产品 + 场景 + 理想生活

**示例**：
- "拥有[产品]，开启[理想生活方式]"
- "[产品]，让[日常场景]变成[美好体验]"
